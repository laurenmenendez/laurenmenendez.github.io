GrowUp!

GrowUp! is a full-stack app that allows parents to digitally document their children privately. Parents can create profiles for each of their children and add features for each child, such as milestones, to memorialize their child's achievements.

Full stack development utilizing commits to Git/Github for version control. I built the RESTful API first to set the foundation for the data I'd be collecting and how I wanted it to be organized, ensuring it would be favorable to a React front-end environment (i.e., setting Django urls such that user id could be pulled from React client routes). Front-end React components and routes allow for easy CRUDing on all data while featuring a smooth and clean UX featuring various React-Bootstrap components like Jumbotrons and Cards. User stories, wireframes, and ERD can be found on the Github repos for this project.

Technologies used: React, Bootstrap, Python, Django (REST framework)

Tell-a-vision

Tell-A-Vision is a full-stack app where you can discuss your favorite TV shows.

Built with 2 other developers in the mob programming style. Full stack development utilizing commits to Git/Github for version control. We began by building out the back-end using Express, Mongoose, and MongoDB with RESTful routes enabling users to CRUD on both posts and comments. All posts/comments are visible, with authentication logic implemented so that users can only edit/delete posts and comments they created. React-Bootstrap components such as Cards and Accordions were utilized to reveal content in a user-friendly and familiar way for interacting with forum apps. User stories, wireframes, and ERD can be found in the Github repos for this project.

Technologies used: React, Bootstrap, Express, Mongoose, MongoDB
